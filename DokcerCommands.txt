6- If you use this command:
   docker run -it --rm openjdk
   you will this docker just a once then it's will deleted.

7- If you use this command:
    docker run -it --rm --name tomcat -p 8888:8080 tomcat:9.0
    you change the name.

8- If you use this command:
    docker rmi 9c7cb1ea80ee
    you delete the image.

There are two types of images:
    1- Pre-build image (Image already exist like tomcat) (Using Docker hub).
    2- Build our image (Image I want to build it). (Write our Dockerfile).
    Here I build the the image:
        FROM openjdk:11
        ARG JAR_FILE=target/*.jar
        COPY ${JAR_FILE} app.jar
        ENTRYPOINT ["java", "-jar", "app.jar"]
